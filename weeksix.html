Creating 3D Objects

Assignment 5

1.Design a nametag for your keychain. Follow the specifications and guildlines.

2.Design a fidget spinner using a roller-skate bearing and M8 nuts.

Key Tag

1. I created a new design file and saved it

2. Started sketch and chose to sketch on X-Y Plane

3. Followed the given guidelines

4. Extruded the outer part of the key tag first to 4mm and then extruded the inner part to 3mm

5. Started a new sketch on the inner part and created a text of my name, and then extruded it afterwards

6. Changed the appearance of my key tag and then exported it

Final Product

  • Key Tag Fusion File

    • Fidget Spinner

    1. I Created a new design file and saved it

    2. Started sketch and chose to sketch on X-Y Plane

    3. Drew 2 circles as the bearing, another big circle to use as guidelines

    4. Drew one arm of the fidget spinner and added hex nuts

    4. Pressed create and used the circular pattern function to make multiple arms

    5. Ended sketch and extruded the necessary fidget spinner part to 8mm

    6. Sketched on the fidget spinner to add designs to make it look nicer

    7. Changed the colour of my fidget spinner
