Desinging an object using revolution

I created a new design file and saved it

I started sketch and chose to sketch on X-Y Plane and then I drew the desired outline of my vase using Spline function.

Using the revolve function, I chose my part and choose the middle line as my axis and revolved it.

Using the shell function, I clicked on the top and put 1mm as the thickness and made the hole for the vase.

Desinging an object using Loft

I created a new design file and saved it

Started sketch and chose to sketch on X-Y Plane

Drew a circle as the base of my bowl and ended the sketch.

Used Offset plane function to create multiple planes

Sketched a bigger circle on the upper plane.

I used the loft function and clicked on the two sketches to make the object solid.

I used the shell function and clicked on the top of the object to turn it into a bowl.

Used revolve function to make the base of the chess piece.
